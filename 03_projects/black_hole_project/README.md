# Black Hole Project



## Project Goal

Learn the physics behind black holes at an introductory level and build a small coding/math project related to gravity near compact objects.



---



# Main Questions

- What is a black hole?

- What is the Schwarzschild radius?

- What is an event horizon?

- How does gravity near a black hole differ from ordinary orbital situations?



---



# Possible Project Directions



## Option A — Schwarzschild Radius Calculator

Build a calculator that computes Schwarzschild radius from mass.



## Option B — Escape Velocity and Collapse

Explore what radius an object would need for escape velocity to equal the speed of light.



## Option C — Orbit / Gravity Comparison

Compare gravity and orbital speeds for normal stars vs compact objects.



## Option D — Research-style explainer

Create a clear writeup with simple equations, diagrams, and a few calculations.



---



# Physics Background

Topics to learn:

- escape velocity

- speed of light as a conceptual threshold in the derivation

- Schwarzschild radius

- event horizon

- limits of Newtonian intuition



---
