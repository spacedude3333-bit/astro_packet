\# Orbit Simulator Project



\## Project Goal

Build a simple simulation of orbital motion to understand gravity, circular motion, and how changing parameters affects orbits.



\---



\# Main Questions

\- How can gravity produce an orbit?

\- What changes if mass increases?

\- What changes if initial velocity changes?

\- What conditions produce circular vs elliptical motion?



\---



\# Physics Background

Important ideas:

\- Newton’s law of gravitation

\- acceleration due to gravity

\- velocity and position updates over time

\- circular vs elliptical motion



Possible formulas:

\- F = G(m1m2 / r^2)

\- a = F / m

\- v = sqrt(GM / r) for circular orbit estimate



\---



\# Minimum Version of the Project

Create a simulation that:

\- models a planet orbiting a star in 2D

\- updates position over small time steps

\- plots the orbit path

\- lets me change mass, radius, or starting velocity



\---



\# Stretch Goals

\- Add multiple planets

\- Add animation

\- Compare circular vs elliptical orbits

\- Plot speed over time

\- Plot distance from star over time
