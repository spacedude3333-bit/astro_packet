# Blackbody / Stellar Radiation Project



## Project Goal

Use coding and graphs to understand blackbody radiation and how star temperature affects emitted light.



---



# Main Questions

- What does a blackbody curve look like?

- How does peak wavelength change with temperature?

- Why do hotter stars appear bluer?

- How does luminosity depend on radius and temperature?



---



# Physics Background

Important laws:

- Wien’s law

- Stefan–Boltzmann law

- blackbody radiation concept



---



# Minimum Version of the Project

Create a project that:

- plots blackbody-like curves or compares peak wavelengths

- shows how hotter stars shift to shorter wavelengths

- explains the relationship between temperature and emitted radiation



---



# Stretch Goals

- Compare several star temperatures on one graph

- Add luminosity calculations

- Connect to HR diagram ideas

- Use real star data if possible



---
